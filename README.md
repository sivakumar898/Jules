# SMS Spam Classifier

This is a simple machine learning project that classifies SMS messages as either "spam" or "ham" (not spam).

## Setup

1. Clone the repository.
2. Install the required Python libraries:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

To run the spam classifier, execute the following command from the root directory:

```bash
python src/spam_classifier.py
```

This will train the model, run predictions on the test set, and print the accuracy and a classification report.

## Dataset

The dataset used is the [SMS Spam Collection Data Set](https://archive.ics.uci.edu/dataset/228/sms+spam+collection) from the UCI Machine Learning Repository.
